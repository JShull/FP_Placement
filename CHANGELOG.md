## [0.3.0] - 2026-02-10

### 0.3.0 Added

- [@JShull](https://github.com/jshull).
  - QuadSurface Placement Support
    - Greedy Placement Quad Algorithm
    - Quad Layout Asset Data Saving
    - QuadSurface Handling in Random Point Placement
  - Secondary Orbital Camera Plane Constraint
    - Added constrained movement plane support for orbital camera controls
  - Measurement + Visualization Tooling (Samples)
    - Measurement Tool Added and Operating
    - Grid Placement Visuals Added
    - Wireframe / Vertex Rendering Updates
  - URP Sample + Render Asset Improvements
    - Added Materials for URP Render Pass Support
    - Volume Profile Added for Sample Environments

### 0.3.0 Changed

- [@JShull](https://github.com/jshull).
  - Placement Interaction Behaviour Updates
    - Improved Single vs Double Click Input Handling
    - Refined Event Binding for Click-Type Actions
    - Implemented Placement State Change Logic
  - Orbital Camera Behaviour Refinement
    - Pan / Orbit Synchronization Improvements
    - Recenter + Navigation Control Updates
    - Multiple Behaviour Passes for Stability and UX
  - Placement Socket System Expansion
    - Socket Capacity Updates
    - Additional Placement States Added
    - Placement Socket Component Iteration

### 0.3.0 Fixed

- [@JShull](https://github.com/jshull).
  - Active Socket Selection Fixes
  - Local Bounds vs World Bounds Placement Bug Fix
  - Sample Stability + General Bug Fix Passes
